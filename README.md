**Webpack**_是一个现代 JavaScript 应用程序的_静态模块打包器\(module bundler\)_。当 webpack 处理应用程序时，它会递归地构建一个

_依赖关系图\(dependency graph\)_，其中包含应用程序需要的每个模块，然后将所有这些模块打包成一个或多个_bundle_。
